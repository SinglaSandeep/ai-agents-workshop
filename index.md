# AI Agents Workshop

This workshop teaches you how to build a local-first multi-agent business assistant, one agent at a time. The use case follows the FoundryIQ and Agent Framework demo pattern: an orchestrator routes questions to HR, Products, and Marketing specialist agents. Each exercise ends with a running version of the application.

## Exercises

This workshop has exercises on:

* Prepare the local development environment
* Build an HR policy agent
* Add a Products knowledge agent
* Add a Marketing knowledge agent
* Add a local orchestrator and web app
* Add observability for Microsoft Foundry and Application Insights

## What You Build

By the end of the workshop, you will have:

* A CLI for testing agents locally
* A FastAPI web app with a simple browser UI
* Local markdown knowledge bases for each specialist agent
* Optional telemetry export to Application Insights
* A path to replace local knowledge with FoundryIQ knowledge bases later

## Prerequisites

For running this workshop you will need:

* Python 3.11 or later
* Visual Studio Code
* PowerShell or another terminal
* Optional: Azure Application Insights for the observability exercise
