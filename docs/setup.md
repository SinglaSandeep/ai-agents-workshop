# Setup & Prerequisites

**Goal:** Get every participant to a working environment with a runnable chat
app and a shared mental model of the Microsoft agent ecosystem before any
building starts. All Azure resources and seed data are **pre-provisioned and
pre-loaded** by your platform team — you verify access and consume them.
