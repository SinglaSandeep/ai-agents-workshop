# Module 3 — Give Agents Access to Tools

**Goal:** Connect Foundry agents to the **MCP tools** you built in Module 2.
You build two specialist agents — **Sales** and **Inventory** — and watch them
call real tools in the Foundry playground.

> **Key concept:** an agent + a tool = a specialist that can both reason *and*
> act on Zava data.
