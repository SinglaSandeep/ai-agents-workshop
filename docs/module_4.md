# Module 4 — Add Knowledge with Foundry IQ

**Goal:** Ground an agent in **knowledge** — unstructured briefs and
post-mortems — using **Foundry IQ**, on top of the MCP tools from Modules 2–3.
You build the **Marketing agent**, which combines both layers in one agent.

> **Key concept:** MCP = tools/actions · Foundry IQ = grounding/knowledge. The
> Marketing agent shows both working together.
