# Module 2 — Build the MCP Tools

**Goal:** Build the three **MCP servers** that give your agents real data —
Sales, Inventory, and Marketing — then run them locally and verify every tool
with the **MCP Inspector**.

> **Key concept:** MCP = the tools/actions layer. An agent calls these tools to
> reach the outside world.
