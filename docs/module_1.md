# Module 1 — Build Your First Agent

**Goal:** Create and test your first Foundry agent — the **Intent Detector** —
so you understand the agent lifecycle (define, deploy, test in the playground)
before adding tools or knowledge.

> **Key concept:** an agent is a model + instructions. This one *reasons* about
> a user's intent; it has no tools yet.
