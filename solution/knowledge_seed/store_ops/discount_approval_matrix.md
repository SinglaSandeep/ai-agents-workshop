# Discount Approval Matrix

This matrix governs who may approve in-store discounts at any Zava store.
All marketing-campaign discounts (e.g., Spring Paint Sale) are pre-approved
and do **not** require additional sign-off; this matrix covers
non-promotional ad-hoc discounts requested by associates, customers, or
contractors.

| Discount band | Approver                        | Notes                                                  |
|---------------|---------------------------------|--------------------------------------------------------|
| 0 – 5%        | Floor associate                 | Logged in POS; no sign-off needed.                     |
| 6 – 10%       | Shift lead                      | Reason code required (price match, damaged box, etc.). |
| 11 – 20%      | Store manager                   | Reason code + manager PIN.                             |
| 21 – 30%      | District manager                | Email to district@zava with item, customer, rationale. |
| 31% or more   | VP Retail Operations            | Written exception only — extraordinary cases.          |

## Price-match guarantee
Associates may match any local competitor's advertised price for an
identical SKU without going up the matrix, provided the proof is attached
to the POS transaction. This is treated as a 0% line and not a discount
under this matrix.

## Pro / contractor pricing
Verified Zava Pro members receive their standard 8% off automatically; this
stacks with campaign promotions only when the campaign brief explicitly
allows it.

## Bulk orders
Orders over $5,000 use a separate quote workflow handled by the Pro Desk;
this matrix does **not** apply.
